# The Fun Folder! <a><img src="assets/controller.png" align="right" height="256" alt="Picture of a video game controller"></a>

There is more to Python than just data analysis!

This folder contains *Introduction to Python for Data Analysis: The Game*. Code can be found in [game.py](game.py).

To run the game, download both [game.py](game.py) and the [assets/](assets/) folder (plus its contents) into the same folder (e.g., place both `game.py` and `assets/` in a folder named `Game`). Run [game.py](game.py) after this (you will need to install package [**pygame-ce**](https://pyga.me/docs/) in addition to packages covered in the workshop).

For ease of use, you may alternatively download a platform-specific `.zip` file containing an executable (`game.app` for MacOS and `game.exe` for Windows). These are [macOS-game.zip](macOS-game.zip) and [windows-game.zip](windows-game.zip). Once downloaded, open the `.zip` file to unzip it, navigate into the folder, and then open the executable to run the game.
